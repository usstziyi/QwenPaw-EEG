# US-009 学习笔记：Source Localization 入门

## 核心概念

### 为什么需要源定位
EEG 记录的是头皮电位，但实际神经活动发生在大脑皮层。容积传导效应导致头皮电位的空间模糊——一个皮层的源会扩散到很多头皮通道。源定位试图重建皮层的电流分布。

### 问题本质
$$X = G \cdot S + \text{noise}$$

- **已知**：X（头皮电位）、G（头模型，正问题）
- **要求**：S（皮层源活动）
- **问题**：这是一个严重欠定的逆问题——源的数量（数千）远大于通道数（几十）

## 前向模型

### 头模型层次
1. **球模型 (Sphere)**：最简单的多层同心球，不需要 MRI
2. **BEM (Boundary Element Method)**：基于 MRI 构建的真实头模型，3 层（大脑、颅骨、头皮）
3. **FEM (Finite Element Method)**：最复杂最精确，考虑组织电导率各向异性

### fsaverage 模板
- 基于 40 个正常人 MRI 的平均脑
- 内置 BEM 模型、皮层表面
- **无需个人 MRI** 即可入门源定位

## 逆问题求解

### MNE (Minimum Norm Estimate)
- 在所有解中选择 L2 范数最小的
- **特点**：最平滑，源偏向表面，深度源被低估
- **输出**：电流密度 (A·m)

### dSPM (dynamic Statistical Parametric Mapping)
- 用噪声协方差归一化 MNE 结果
- **特点**：深度补偿，Z-score 方便统计
- **输出**：Z-score

### sLORETA (standardized LORETA)
- 零定位误差（理想单源条件下）
- **特点**：比 MNE 更准，最常用的标准化方法
- **输出**：Z-score（类似）

### 正则化参数 λ²
- 控制先验（平滑）和拟合数据之间的平衡
- λ² = 1/SNR²，SNR 通常设为 3（λ² = 1/9）
- 越大 → 越平滑，越保守

## 参数调优

### loose
- 0：强制源方向严格垂直于皮层
- 1：允许源自由朝向
- 0.2-0.4：常用值，大部分垂直但有自由度

### depth
- 0：无深度加权（表面源占优势）
- 1：完全深度加权（抵消表面偏向）
- 0.8：MNE 默认，常用值

## 验证

源定位结果**必须谨慎解读**——它是统计推断，不是测量。

验证方法：
1. 已知功能区激活（听觉→颞叶，视觉→枕叶）
2. 与 fMRI 结果对比（如有）
3. 多方法一致性（MNE/dSPM/sLORETA 是否一致）

## 关键经验

1. **fsaverage 是入门神器**：不需要 MRI，30 分钟跑通全流程
2. **BEM 需要 MRI + FreeSurfer**：进阶必备，精度显著提高
3. **源定位不是万能的**：浅源定位最好，深源（如海马）几乎不可能
4. **噪声协方差要谨慎估计**：用 baseline 时段（事件前安静期）
5. **别只看一个时间点**：画时间动画 (time_viewer=True) 看动态
6. **sLORETA 通常最靠谱**：相比 MNE 定位更准
7. **球模型是快速替代**：不需要下载 fsaverage，精度差但速度快

## 常见错误

| 错误 | 表现 | 解决 |
|------|------|------|
| 没下载 fsaverage | FileNotFoundError | `mne.datasets.fetch_fsaverage()` |
| 通道与 trans 不匹配 | 源全在一个不相关的位置 | 检查 montage 和 trans |
| λ² 太小 | 噪声也被放大了，源分布杂乱 | 增大 λ² |
| mindist 太小 | 源出现在颅骨外 | 设 mindist≥5 |
| 忘记设 eeg=True | 用 MEG 通道做 EEG 源定位 | 确认 `make_forward_solution` 参数 |
