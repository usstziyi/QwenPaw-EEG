# US-004 学习笔记：预处理

## 预处理标准顺序

```
1. 陷波 (Notch)          → 去 50/60 Hz 工频
2. 带通滤波 (Bandpass)    → 去漂移和肌电高频
3. 降采样 (Resample)     → 减少数据量
4. 重参考 (Re-reference) → 选择电位参考点
5. 坏导插值 (Interpolate) → 修复坏通道
```

## 滤波

### 截止频率选择

| 频段 | 范围 | 保留策略 |
|------|------|----------|
| Delta | 0.5-4 Hz | 只分析慢波时保留 |
| Theta | 4-8 Hz | 认知研究常见 |
| Alpha | 8-13 Hz | BCI 运动想象重点 |
| Beta | 13-30 Hz | 运动/感觉 |
| Gamma | 30+ Hz | 需要高采样率（>250Hz） |

### 陷波 (Notch)

- 频率：中国/欧洲 50 Hz，北美 60 Hz
- **不要在陷波后再降采样到低于 2×工频**（会导致滤波失效）
- MNE 的 `notch_filter` 默认是零相位 IIR

### 滤波器类型

- **FIR（默认）**：稳定、零相位、但滤波器可能很长
- **IIR**：更快但可能引入相位延迟
- MNE 默认 `phase='zero'` 即零相位滤波，无需担心延迟

### 参数调优

- `l_trans_bandwidth`：越小滤波越陡但越长，默认 'auto' 通常够用
- `filter_length`：'auto' 时自动选择，长数据用 10s 窗口
- 滤波前如果数据有坏段（大幅跳变），先标记 Annotation 或用 `raw.crop()`

## 降采样

### Nyquist 定理
目标采样率 **必须 > 2 × 最高感兴趣频率**。

例：关心到 40 Hz → 降采样到 ≥ 100 Hz（留余量）。

**典型设置：**
- 原始 500-1000 Hz → 目标 200-250 Hz（ERP/时频）
- 原始 500-1000 Hz → 目标 500+ Hz（高频 gamma）

### 一步到位
`raw.resample(sfreq)` 内部自动做了低通滤波 + 降采样，可以一步完成。

## 重参考

### 平均参考
- **最常用**，假设头表电位总和为零
- 通道够多（≥32 导）时效果最好
- 缺点：对局部大活动会被「扩散」到其他通道

### 乳突参考
- 常用于临床 ERP 研究（如 P300）
- 乳突电极本身也有脑电活动，不是真正的「零」

### REST 参考
- 理论上更优（近似无穷远参考）
- 需要头模型，用 `mne.set_eeg_reference(..., ref_channels='REST')`

## 坏导处理

### 检测
- 肉眼 + raw.plot() 是最可靠的方法
- 自动化：`find_bad_channels_maxwell()`（需要 Maxwell 滤波）或基于方差/峰度的自定义逻辑

### 插值
- `raw.interpolate_bads()` 使用球面样条（spherical spline）
- 前提：有 montage（电极位置）
- 坏导太多（>10%）时不建议插值，直接剔除通道

## 关键经验

1. **不要反复滤波**：每次滤波都会有边界效应，MNE 会自动处理边界但多次累积可能有问题
2. **先滤波再降采样**：或者用 `resample()` 一步完成
3. **滤波前检查数据**：大跳变会导致滤波器「振铃」
4. **坏导标记要准确**：明明好的通道别误标，坏了的不标后续会影响 ICA
5. **预处理后始终存一份 .fif**：心血别白费
